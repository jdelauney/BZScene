TBZBitmapAlphaSetMode BZGraphic

Définit comment la Canal Alpha doit être définit dans un bitmap :

  • asmDefault : Utilise le canal alpha non modifié du bitmap par defaut

  • asmAlphaFromIntensity : La valeur du canal alpha est déduite des autres intensité des composants RVB (plus lumineux, plus opaque)

  • asmSuperBlackTransparent : Les pixels avec une couleur RVB de (0, 0, 0) sont complètement transparent, les autres reste inchangés

  • asmLuminance : La valeur de luminance est calculée pour chaque pixel

  • asmLuminanceSqrt : Identique à bamLuminance mais avec un Sqrt (Luminance)

  • asmOpaque : Le canal alpha est uniformément réglé sur 255


Note : Luminance = Intensité , mais plus précis cf : fonction TBZBitmapColorFilters.SetAlpha
